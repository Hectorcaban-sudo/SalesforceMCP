# pubsub_api.proto

This file is the official Salesforce Pub/Sub API gRPC protobuf definition.

Download the latest version from the Salesforce developer site:

    https://github.com/developerforce/pub-sub-api/blob/main/pubsub_api.proto

Save it in this directory as `pubsub_api.proto` before running `npm start`.
The subscriber code loads it via `@grpc/proto-loader` at startup.
